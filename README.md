📊 Projeto de Análise de Vendas com SQL

📖 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de simular a rotina de um Analista de Dados em uma empresa de e-commerce.

A proposta consiste em criar um banco de dados relacional utilizando MySQL, realizar consultas SQL para responder perguntas de negócio e, posteriormente, construir um dashboard no Power BI para visualização dos indicadores.

O projeto faz parte do meu portfólio profissional voltado para a área de Dados.


🎯 Objetivos

- Modelar um banco de dados relacional.
- Praticar consultas SQL do nível básico ao intermediário.
- Realizar análises de vendas.
- Gerar indicadores para tomada de decisão.
- Desenvolver um dashboard em Power BI.


🛠 Tecnologias utilizadas

- MySQL Workbench
- SQL
- Power BI *(em desenvolvimento)*
- Git
- GitHub
